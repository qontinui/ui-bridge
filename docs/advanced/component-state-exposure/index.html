Component State Exposure

Component State Exposure allows you to expose internal component state to UI Bridge, making it accessible for inspection and automation.

Overview​

State exposure enables:

  • Access to React component state
  • Computed/derived state visibility
  • Form state inspection
  • Custom state hooks

Basic State Exposure​

React Hook​

import { useExposeState } from '@anthropic/ui-bridge/react';

function Counter() {
  const [count, setCount] = useState(0);

  // Expose count to UI Bridge
  useExposeState('counter', { count });

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(c => c + 1)}>Increment</button>
    </div>
  );
}

Access Exposed State​

// From automation client
const state = await client.getComponentState('counter');
console.log(state.count); // 5

State Types​

Primitive Values​

useExposeState('settings', {
  darkMode: true,
  language: 'en',
  fontSize: 16,
});

Objects and Arrays​

useExposeState('cart', {
  items: [
    { id: 1, name: 'Product', quantity: 2 },
    { id: 2, name: 'Another', quantity: 1 },
  ],
  total: 99.99,
});

Computed State​

function ShoppingCart({ items }) {
  const subtotal = items.reduce((sum, item) => sum + item.price * item.quantity, 0);
  const tax = subtotal * 0.08;
  const total = subtotal + tax;

  useExposeState('cart-totals', {
    subtotal,
    tax,
    total,
    itemCount: items.length,
  });

  return <div>...</div>;
}

Form State​

Expose Form Values​

function LoginForm() {
  const [formData, setFormData] = useState({
    email: '',
    password: '',
    rememberMe: false,
  });

  const [errors, setErrors] = useState({});
  const [isSubmitting, setIsSubmitting] = useState(false);

  useExposeState('login-form', {
    values: formData,
    errors,
    isSubmitting,
    isValid: Object.keys(errors).length === 0,
  });

  return <form>...</form>;
}

Access Form State​

// Check form state before submission
const formState = await client.getComponentState('login-form');

if (formState.isValid && !formState.isSubmitting) {
  await client.executeAction('submit-btn', 'click');
}

State Namespacing​

Component Scoping​

// In UserProfile component
useExposeState('user-profile.details', {
  name: user.name,
  email: user.email,
});

useExposeState('user-profile.preferences', {
  theme: user.preferences.theme,
  notifications: user.preferences.notifications,
});

Access Namespaced State​

const details = await client.getComponentState('user-profile.details');
const prefs = await client.getComponentState('user-profile.preferences');

// Or get all under namespace
const userProfile = await client.getComponentState('user-profile.*');

State Watchers​

Watch for Changes​

// From automation client
const unwatch = client.watchState('cart', (state, prevState) => {
  console.log(`Cart updated: ${prevState.total} -> ${state.total}`);
});

// Later: unwatch()

Wait for State​

// Wait for specific state condition
await client.waitForState('cart', (state) => state.items.length > 0);

// With timeout
await client.waitForState('login', (state) => state.isAuthenticated, {
  timeout: 5000,
});

Selective Exposure​

Only Expose What's Needed​

function UserDashboard({ user }) {
  // Don't expose sensitive data
  useExposeState('user', {
    id: user.id,
    name: user.name,
    // Don't include: user.password, user.ssn, user.creditCard
  });
}

Conditional Exposure​

function AdminPanel({ isAdmin, adminData }) {
  useExposeState('admin', isAdmin ? {
    userCount: adminData.userCount,
    revenue: adminData.revenue,
  } : null);
}

State Metadata​

Add Metadata to State​

useExposeState('order', {
  status: order.status,
  total: order.total,
}, {
  metadata: {
    lastUpdated: Date.now(),
    source: 'api',
  },
  description: 'Current order state',
});

Vue Integration​

<script setup>
import { ref } from 'vue';
import { useExposeState } from '@anthropic/ui-bridge-vue';

const count = ref(0);

useExposeState('counter', () => ({
  count: count.value,
}));
</script>

Angular Integration​

@Component({...})
export class CounterComponent {
  count = 0;

  constructor(private uiBridge: UIBridgeService) {
    this.uiBridge.exposeState('counter', () => ({
      count: this.count,
    }));
  }
}

State Snapshots​

Get All Exposed State​

const snapshot = await client.getStateSnapshot();

console.log(snapshot);
// {
//   'counter': { count: 5 },
//   'cart': { items: [...], total: 99.99 },
//   'login-form': { values: {...}, isValid: true },
// }

Diff State​

const before = await client.getStateSnapshot();
await client.executeAction('add-to-cart-btn', 'click');
const after = await client.getStateSnapshot();

const diff = client.diffState(before, after);
// { 'cart.items': { added: [...] }, 'cart.total': { old: 99.99, new: 149.99 } }

Best Practices​

  1. Expose Only Needed State - Don't expose everything
  2. Never Expose Sensitive Data - Passwords, tokens, PII
  3. Use Namespacing - Organize state logically
  4. Keep State Flat - Avoid deeply nested structures
  5. Update Synchronously - Ensure state reflects actual component state
  6. Document Exposed State - List what state is available
