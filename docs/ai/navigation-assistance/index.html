Navigation Assistance

Navigation Assistance helps AI agents understand how to navigate between pages and reach target states within your application.

Overview​

Navigation Assistance provides:

  • Path finding between pages
  • Breadcrumb tracking
  • Navigation hints and suggestions
  • State transition mapping

Usage​

Find Navigation Path​

import { NavigationAssistant } from '@anthropic/ui-bridge';

const assistant = new NavigationAssistant(registry);

// Find how to reach a page
const path = await assistant.findPath('checkout page');

console.log(path);
// {
//   steps: [
//     { action: 'click', target: 'cart-icon', description: 'Open shopping cart' },
//     { action: 'click', target: 'checkout-btn', description: 'Proceed to checkout' }
//   ],
//   confidence: 0.95
// }

Get Navigation Hints​

const hints = await assistant.getHints();

console.log(hints);
// {
//   currentPage: '/products',
//   breadcrumbs: ['Home', 'Products'],
//   availableNavigations: [
//     { label: 'Home', path: '/', element: 'nav-home' },
//     { label: 'Cart', path: '/cart', element: 'cart-icon' },
//     { label: 'Account', path: '/account', element: 'account-menu' }
//   ],
//   suggestedActions: [
//     'Add product to cart',
//     'Filter by category',
//     'Sort by price'
//   ]
// }

Track Breadcrumbs​

// Automatically tracks navigation history
assistant.enableBreadcrumbTracking();

// Get current breadcrumbs
const breadcrumbs = assistant.getBreadcrumbs();
// ['Home', 'Products', 'Electronics', 'Phones']

// Navigate back
await assistant.navigateBack(2);  // Goes to 'Products'

Navigation Map​

Define your application's navigation structure:

const assistant = new NavigationAssistant(registry, {
  navigationMap: {
    'home': {
      path: '/',
      links: ['products', 'cart', 'account'],
    },
    'products': {
      path: '/products',
      parent: 'home',
      links: ['product-detail', 'cart'],
    },
    'cart': {
      path: '/cart',
      parent: 'home',
      links: ['checkout', 'products'],
    },
    'checkout': {
      path: '/checkout',
      parent: 'cart',
      links: ['confirmation'],
      requiresAuth: true,
    },
  },
});

Auto-Discovery​

Let the assistant learn navigation patterns:

const assistant = new NavigationAssistant(registry, {
  autoDiscover: true,
  discoveryOptions: {
    followLinks: true,
    maxDepth: 3,
    excludePatterns: ['/admin/*', '/api/*'],
  },
});

// Start discovery
await assistant.discoverNavigation();

// Export discovered map
const map = assistant.exportNavigationMap();

Path Finding​

Simple Path​

const path = await assistant.findPath('settings');
// Returns shortest path to settings page

Conditional Path​

const path = await assistant.findPath('checkout', {
  requireAuth: true,
  preferredRoute: 'through-cart',
});

// Returns path that ensures user is logged in

Multi-Step Path​

const path = await assistant.findPath('order-confirmation', {
  from: 'home',
  through: ['cart', 'checkout'],
});

Navigation State​

Track the current navigation state:

interface NavigationState {
  currentPage: string;
  previousPage: string;
  breadcrumbs: string[];
  canGoBack: boolean;
  canGoForward: boolean;
  history: NavigationEntry[];
}

const state = assistant.getNavigationState();

Event Handling​

Listen for navigation events:

assistant.on('navigate', (event) => {
  console.log(`Navigated from ${event.from} to ${event.to}`);
});

assistant.on('pathNotFound', (event) => {
  console.log(`Cannot find path to ${event.target}`);
});

assistant.on('breadcrumbUpdate', (breadcrumbs) => {
  console.log('Breadcrumbs:', breadcrumbs);
});

Integration with AI​

Context for AI Agents​

async function provideNavigationContext() {
  const hints = await assistant.getHints();
  const state = assistant.getNavigationState();

  return {
    currentLocation: state.currentPage,
    canNavigateTo: hints.availableNavigations.map(n => n.label),
    suggestedActions: hints.suggestedActions,
    breadcrumbs: state.breadcrumbs,
  };
}

AI-Driven Navigation​

async function navigateWithAI(destination: string) {
  const path = await assistant.findPath(destination);

  if (!path) {
    throw new Error(`Cannot find path to ${destination}`);
  }

  for (const step of path.steps) {
    await intentExecutor.execute(`${step.action} ${step.target}`);
    await waitForNavigation();
  }
}

Configuration​

const assistant = new NavigationAssistant(registry, {
  trackHistory: true,        // Track navigation history
  maxHistoryLength: 50,      // Max history entries
  breadcrumbSelector: '.breadcrumb', // Auto-detect breadcrumbs
  navigationSelector: 'nav', // Navigation element selector
  linkPatterns: [            // Patterns for navigation links
    /^\/[a-z-]+$/,
    /^#[a-z-]+$/,
  ],
});

Best Practices​

  1. Define Navigation Map - Explicit maps are more reliable than auto-discovery
  2. Track Breadcrumbs - Essential for understanding user context
  3. Handle Auth Requirements - Mark pages that require authentication
  4. Provide Hints - Give AI agents clear navigation options
  5. Test Paths - Verify that discovered paths are actually navigable
