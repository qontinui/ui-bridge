Performance Metrics

UI Bridge collects performance metrics to help you understand and optimize automation performance.

Overview​

Performance metrics provide:

  • Action timing and latency
  • Element discovery performance
  • Network request tracking
  • Memory usage monitoring
  • Custom metric collection

Enabling Metrics​

import { configureMetrics } from '@anthropic/ui-bridge';

configureMetrics({
  enabled: true,
  collectInterval: 1000,  // Collect every second
  historySize: 1000,      // Keep last 1000 entries
});

Built-in Metrics​

Action Metrics​

import { getActionMetrics } from '@anthropic/ui-bridge';

const metrics = getActionMetrics();

console.log(metrics);
// {
//   totalActions: 150,
//   successRate: 0.98,
//   avgDuration: 245,
//   p50Duration: 180,
//   p95Duration: 450,
//   p99Duration: 890,
//   byAction: {
//     click: { count: 80, avgDuration: 120 },
//     type: { count: 50, avgDuration: 350 },
//     scroll: { count: 20, avgDuration: 200 },
//   },
// }

Discovery Metrics​

import { getDiscoveryMetrics } from '@anthropic/ui-bridge';

const metrics = getDiscoveryMetrics();

Real-Time Monitoring​

Subscribe to Metrics​

import { subscribeToMetrics } from '@anthropic/ui-bridge';

const unsubscribe = subscribeToMetrics((metrics) => {
  console.log('Current metrics:', metrics);
});

// Later: unsubscribe()

Metric Events​

import { onMetricEvent } from '@anthropic/ui-bridge';

onMetricEvent('slowAction', (event) => {
  console.log(`Slow action detected: ${event.action} took ${event.duration}ms`);
});

onMetricEvent('highErrorRate', (event) => {
  console.log(`Error rate above threshold: ${event.rate}`);
});

Custom Metrics​

Record Custom Metrics​

import { recordMetric } from '@anthropic/ui-bridge';

recordMetric('custom.checkout.time', 2500);
recordMetric('custom.cart.items', 3);

Define Custom Metric Types​

import { defineMetric } from '@anthropic/ui-bridge';

defineMetric({
  name: 'checkout.conversion',
  type: 'gauge',
  description: 'Checkout conversion rate',
  unit: 'percent',
  labels: ['variant'],
});

recordMetric('checkout.conversion', 0.65, { variant: 'A' });

Histograms​

Track distribution of values:

import { createHistogram } from '@anthropic/ui-bridge';

const actionDuration = createHistogram({
  name: 'action.duration',
  buckets: [10, 50, 100, 250, 500, 1000, 2500, 5000],
});

// Record values
actionDuration.observe(150);
actionDuration.observe(280);

// Get distribution
console.log(actionDuration.getStats());
// { count: 2, sum: 430, avg: 215, p50: 150, p95: 280 }

Export Formats​

Prometheus​

import { exportPrometheus } from '@anthropic/ui-bridge';

const metrics = exportPrometheus();
// Returns Prometheus text format

JSON​

import { exportMetrics } from '@anthropic/ui-bridge';

const metrics = exportMetrics('json');
// Returns structured JSON

OpenTelemetry​

import { configureOTelExporter } from '@anthropic/ui-bridge';

configureOTelExporter({
  endpoint: 'http://collector:4318/v1/metrics',
  interval: 10000,
});

Alerting​

Configure Alerts​

import { configureAlerts } from '@anthropic/ui-bridge';

configureAlerts({
  rules: [
    {
      metric: 'action.errorRate',
      condition: '> 0.05',
      duration: '1m',
      handler: (alert) => {
        notificationService.send(`High error rate: ${alert.value}`);
      },
    },
    {
      metric: 'action.p95Duration',
      condition: '> 1000',
      handler: (alert) => {
        console.warn(`Slow actions detected: p95=${alert.value}ms`);
      },
    },
  ],
});

Configuration​

configureMetrics({
  enabled: true,
  collectInterval: 1000,
  historySize: 1000,

  // What to collect
  collect: {
    actions: true,
    discovery: true,
    network: true,
    memory: true,
  },

  // Thresholds
  thresholds: {
    slowAction: 1000,
    highErrorRate: 0.05,
    highMemory: 100 * 1024 * 1024,  // 100MB
  },

  // Export
  export: {
    format: 'prometheus',
    endpoint: '/metrics',
    interval: 15000,
  },
});

Best Practices​

  1. Set Baselines - Establish performance baselines before optimization
  2. Monitor Percentiles - p95/p99 are more useful than averages
  3. Alert on Anomalies - Set alerts for significant deviations
  4. Export to Backend - Send metrics to a time-series database
  5. Use Labels - Add context with metric labels
  6. Regular Review - Review metrics dashboards regularly
