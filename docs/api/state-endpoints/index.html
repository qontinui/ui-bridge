State API Endpoints

This page documents the state management REST API endpoints provided by UI Bridge.

Base URL​

All state endpoints are prefixed with /__ui-bridge__/state/.

Endpoints​

Get Current State​

Get the current state machine state.

GET /__ui-bridge__/state/current

Response:

{
  "success": true,
  "state": {
    "id": "logged-in",
    "description": "User is authenticated",
    "enteredAt": 1705312200000,
    "indicators": {
      "element-exists:#user-menu": true,
      "url-matches:/dashboard/": true
    }
  }
}

List All States​

List all defined states.

GET /__ui-bridge__/state/list

Response:

{
  "success": true,
  "states": [
    {
      "id": "logged-out",
      "description": "User not authenticated",
      "indicators": ["element-exists:#login-form"]
    },
    {
      "id": "logged-in",
      "description": "User authenticated",
      "indicators": ["element-exists:#user-menu"]
    },
    {
      "id": "dashboard",
      "description": "Main dashboard view",
      "indicators": ["url-matches:/dashboard/"]
    }
  ]
}

Get State Details​

Get details about a specific state.

GET /__ui-bridge__/state/:stateId

Response:

{
  "success": true,
  "state": {
    "id": "logged-in",
    "description": "User is authenticated",
    "indicators": [
      { "type": "element-exists", "selector": "#user-menu" },
      { "type": "element-not-exists", "selector": "#login-form" }
    ],
    "transitions": {
      "outgoing": ["logged-in -> dashboard", "logged-in -> logged-out"],
      "incoming": ["logged-out -> logged-in"]
    }
  }
}

Validate State​

Check if a state's indicators match current UI.

POST /__ui-bridge__/state/validate

Request Body:

{
  "stateId": "logged-in"
}

Response:

{
  "success": true,
  "valid": true,
  "indicators": {
    "element-exists:#user-menu": { "expected": true, "actual": true },
    "element-not-exists:#login-form": { "expected": true, "actual": true }
  }
}

Navigate to State​

Navigate to a target state.

POST /__ui-bridge__/state/navigate

Request Body:

{
  "target": "dashboard",
  "variables": {
    "email": "user@example.com",
    "password": "secret"
  },
  "options": {
    "timeout": 30000
  }
}

Response:

{
  "success": true,
  "path": ["logged-out -> logged-in", "logged-in -> dashboard"],
  "currentState": "dashboard",
  "duration": 2500
}

Execute Transition​

Execute a specific state transition.

POST /__ui-bridge__/state/transition

Request Body:

{
  "transition": "logged-out -> logged-in",
  "variables": {
    "email": "user@example.com",
    "password": "secret"
  }
}

Response:

{
  "success": true,
  "from": "logged-out",
  "to": "logged-in",
  "duration": 1200
}

Find Path​

Find a path between two states.

POST /__ui-bridge__/state/find-path

Request Body:

{
  "from": "logged-out",
  "to": "checkout"
}

Response:

{
  "success": true,
  "path": [
    "logged-out -> logged-in",
    "logged-in -> products",
    "products -> cart",
    "cart -> checkout"
  ],
  "steps": 4
}

Get Discovered States​

Get states discovered by dynamic state discovery.

GET /__ui-bridge__/state/discovered

Response:

{
  "success": true,
  "states": [
    {
      "id": "state-abc123",
      "signature": {
        "url": "/products",
        "domFingerprint": "abc123"
      },
      "discoveredAt": 1705312200000,
      "visitCount": 5
    }
  ]
}

Get Component State​

Get exposed component state.

GET /__ui-bridge__/state/component/:componentId

Response:

{
  "success": true,
  "componentId": "counter",
  "state": {
    "count": 5,
    "isLoading": false
  },
  "lastUpdated": 1705312200000
}

Get All Component States​

Get all exposed component states.

GET /__ui-bridge__/state/components

Response:

{
  "success": true,
  "components": {
    "counter": { "count": 5 },
    "cart": { "items": [], "total": 0 },
    "user": { "name": "John", "email": "john@example.com" }
  }
}

Watch State Changes​

Subscribe to state change events (Server-Sent Events).

GET /__ui-bridge__/state/watch

Response (SSE stream):

event: stateChange
data: {"from": "logged-out", "to": "logged-in", "timestamp": 1705312200000}

event: componentUpdate
data: {"component": "cart", "state": {"items": [...]}, "timestamp": 1705312201000}

Export State Machine​

Export the state machine configuration.

GET /__ui-bridge__/state/export

Query Parameters:

ParameterTypeDescription
formatstringjson or mermaid

Response (JSON):

{
  "success": true,
  "stateMachine": {
    "states": {...},
    "transitions": {...},
    "initialState": "logged-out"
  }
}

Response (Mermaid):

{
  "success": true,
  "diagram": "stateDiagram-v2\n  [*] --> logged_out\n  logged_out --> logged_in: login\n  ..."
}

Import State Machine​

Import a state machine configuration.

POST /__ui-bridge__/state/import

Request Body:

{
  "stateMachine": {
    "states": {
      "logged-out": {...},
      "logged-in": {...}
    },
    "transitions": {...}
  },
  "merge": false
}

Response:

{
  "success": true,
  "imported": {
    "states": 5,
    "transitions": 8
  }
}

Error Responses​

All endpoints return errors in this format:

{
  "success": false,
  "error": {
    "code": "STATE_NOT_FOUND",
    "message": "State 'unknown' does not exist",
    "details": {
      "requestedState": "unknown",
      "availableStates": ["logged-out", "logged-in", "dashboard"]
    }
  }
}

Common Error Codes​

CodeDescription
STATE_NOT_FOUNDRequested state doesn't exist
TRANSITION_NOT_FOUNDRequested transition doesn't exist
NO_PATH_FOUNDCannot find path between states
TRANSITION_FAILEDTransition execution failed
INVALID_STATE_CONFIGInvalid state machine configuration
COMPONENT_NOT_FOUNDComponent state not exposed
